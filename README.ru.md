# DiskUsage – простой анализатор занятого места на диске для macOS

**Язык:** [English](README.md) | [Русский](README.ru.md)

DiskUsage — небольшое нативное приложение для macOS на Swift/SwiftUI.  
Помогает быстро понять, какие папки занимают больше всего места на диске.

> Я развиваю проект в свободное время и коплю на подписку Apple Developer
> Program (99 USD в год), чтобы подписывать и нотарифицировать приложение,
> а в будущем выложить его в Mac App Store.  
> Подробнее см. раздел «Поддержка».

---

## Возможности

- Сканирование:
  - домашней папки пользователя;
  - всего диска `/` (только по явной команде);
  - любой выбранной папки через стандартный диалог.
- Группировка по крупным каталогам (первый уровень под выбранным корнем).
- Drill-down навигация:
  - клик по строке — переход на уровень глубже;
  - кнопка «Назад» — подняться на уровень выше.
- Отображение:
  - размер папок в удобных единицах (Б / КБ / МБ / ГБ / ТБ);
  - процент от общего объёма в рамках выбранного корня;
  - список папок, к которым нет доступа из-за прав.
- Асинхронное сканирование — интерфейс не «подвисает».
- Локализация:
  - русский и английский;
  - при любой не русской локали системы используется английский.

---

## Технологии

- macOS, Swift 5+
- SwiftUI
- MVVM:
  - `DiskScannerViewModel` как ViewModel;
  - `DiskUsageService` как сервис сканирования.
- Swift Concurrency:
  - фоновое сканирование через `Task.detached`;
  - обновление UI только на `MainActor`.
- Локализация через String Catalog (`Localizable.xcstrings`).
