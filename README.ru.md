# DiskUsage

**Languages:** [English](README.md) | [Русский](README.ru.md)

Быстрый и приватный анализатор диска для macOS на SwiftUI.

## Зачем DiskUsage
- Быстро видно лишнее: дерево или круговая диаграмма сразу показывают «пожирателей» места.
- Действия под рукой: открыть в Finder, скопировать путь, отправить в корзину.
- Работает где угодно: домашняя папка, весь диск (`/`) или выбранная директория.
- Минимум хлопот: локальное сканирование, без аккаунтов и трекинга.

## Основное
- Панель диска с общим/занятым/свободным объёмом.
- Сортируемое дерево с полосками размеров.
- Sunburst для быстрого обзора горячих точек.
- Настройки: режим по умолчанию, язык, подтверждение удаления, скрытые файлы.
- Локализация: английский и русский.

## Требования
- macOS 14.0+
- Xcode 15+

## Запуск
1. Откройте `DiskUsage.xcodeproj` в Xcode.
2. Соберите и запустите.
3. Если сканируете защищённые области, включите **Full Disk Access**: System Settings → Privacy & Security.
