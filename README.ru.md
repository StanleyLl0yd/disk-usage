# DiskUsage

**Язык:** [English](README.md) | [Русский](README.ru.md)

Нативное приложение для macOS для анализа занятого места на диске, написанное на SwiftUI.

> Я развиваю проект в свободное время и собираю средства на подписку Apple
> Developer Program (99 USD в год), чтобы подписывать, нотарифицировать и в
> перспективе опубликовать приложение в Mac App Store. См. раздел **Поддержка**,
> если хотите помочь.

## Возможности

- **Два режима визуализации:**
  - Древовидный список с раскрывающимися папками
  - Круговая диаграмма (в стиле DaisyDisk)
- **Панель информации о диске** — общий/используемый/свободный объём с цветовым прогрессом
- **Сканирование:**
  - Домашней папки
  - Всего диска (/)
  - Произвольной папки
- **Операции с файлами:**
  - Показать в Finder
  - Скопировать путь
  - Переместить в корзину (с пересчётом объёма)
- **Настройки:**
  - Режим отображения по умолчанию
  - Язык (Системный/English/Русский)
  - Подтверждение удаления
  - Показывать скрытые файлы
- **Полная локализация:** английский и русский

## Требования

- macOS 14.0+
- Xcode 15+
- Полный доступ к диску (для сканирования системных папок)

## Установка

1. Откройте `DiskUsage.xcodeproj` в Xcode
2. Соберите и запустите приложение
3. Выдайте Full Disk Access в Настройки системы → Конфиденциальность и безопасность → Полный доступ к диску

## Файлы

```
DiskUsageApp.swift       — точка входа приложения
ContentView.swift        — главный контейнер UI
TreeView.swift           — древовидная визуализация
SunburstView.swift       — круговая визуализация
DiskScanner.swift        — сканер файловой системы
DiskScannerViewModel.swift — управление состоянием
FolderUsage.swift        — модель данных
Settings.swift           — модель настроек
SettingsView.swift       — UI настроек
Utilities.swift          — вспомогательные функции форматирования
Localizable.xcstrings    — локализация
DiskUsage.entitlements   — entitlements приложения
```
